Fonctionalites :

Retour arriere/avant des figures avec ctrl-z/ctrl-y(All States)
Annulation d'une selection avec echap(Zone Select State et Point Select State)
Finalisation de la creation d'un polygone avec echap (Create State)
Modification de la position d'un point avec la souris apres avoir clicke dessus(Point Select State)
Modification de la position de plusiseurs figures avec la souris apres les avoirs selectionner et bouger la zone de selection avc la souris(Zone Select State)
Suppression d'un point avec echap apres l'avoir selectionner(Point Select State)
Suppression de plusiseurs figures avec echap apres les avoirs selectionner(Zone Select State)
Copie d'une selection de figures avec ctrl-c (Zone Select State)
Collage a la position de la souris d'une selection de figures avec ctrl-v(Zone Select State)
Creation Point,Segment,Cercle,Polygone(CreateState)
Bouton C pour clear les figures
Sauvegarde des figures a la fermeture de l'application
Chargement des figures a son ouverture
